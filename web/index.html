Beschreibung des Semesterprojektes:

Django Webapplikation (features)

  • User Authentifizierung

    1. User meldet sich an
    2. User bekommt eine Bestaetigungsmail
    3. User ist Authentifiziert und kann sich anmelden
    4. (Passwort vergessen ebenfalls implementiert)

  • OAuth Moves App

    1. User verbindet sich mit seiner Moves-App
    2. User bekommt von Authentifizierungs-Service der Moves-Applikation eine Pin angezeigt
    3. Diese Pin traegt der User in seine Handy-App ein
    4. Automatischer redirect zur Webapplikation (django)

  • Importieren der Moves-Daten

    1. Der User kann die Daten in die Webapplikation (in seinen User-Context) importieren. (die Api von Moves erlaubt nur Abfragen ueber bestimmte Zeitbereiche (max 30 Tage))

  • Darstellen der Daten der Moves-App

    1. Der User kann sich monatliche Tabellen-Uebersicht seiner Moves-Daten anzeigen.

      Tabellenansicht der letzten 30 Tage.

    2. Der User kann sich eine Kartenansicht des Tages mit Bewegungsablaeufen anzeigen lassen.

      Kartenansicht fuer das Datum 1-11-2018
